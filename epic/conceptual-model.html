Conceptual Model

User

  • User Id (PK)
  • User Hash
  • Date of Birth
  • User Email

Content

  • Content Name (PK)
  • Content Description
  • Genre
  • User Feedback (FK)

Relationship

  • A profile can watch multiple varieties of content (1 to 1)
  • Users can provide feedback to individual pieces of content (m to n)
  • Feed back is received by multiple users (m to n)
